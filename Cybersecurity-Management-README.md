# Security Management Briefings & Analysis — CMIT 425 | UMGC
Verified coursework from the University of Maryland Global Campus.  
Topics covered: Organizational security roles, cryptography, VPN protocols, access controls, security auditing, physical security, and software development security.

---

## 🗂️ Course Context

All briefings in this course were developed in the role of a **Cybersecurity Management professional** advising organizational leadership on security policy, risk management, and compliance. Each briefing addresses a real-world security management scenario with research-backed analysis and actionable recommendations.

| Briefing | Topic | Key Concepts |
|----------|-------|-------------|
| [Briefing 2 — RACI Matrix & Organizational Roles](Briefing2-RACI-Matrix-Organizational-Roles.md) | Defining security roles and responsibilities across the organization | CEO, CFO, CIO, CPO, CSO, RACI framework |

## 🛠️ Skills Demonstrated

- **Security Governance** — Defined organizational roles and responsibilities using RACI frameworks
- **Cryptography Analysis** — Evaluated symmetric and asymmetric encryption methods and real-world attack vectors
- **Network Security** — Compared VPN protocols for securing remote workforce environments
- **Access Control Policy** — Analyzed access control models and insider threat scenarios for WFH environments
- **Security Auditing** — Planned and scoped enterprise-level remote work device audits
- **Physical Security** — Designed physical security controls and controlled zone frameworks
- **Secure SDLC** — Integrated security into software development using CMMI Level 3 and OWASP standards
- **Risk Management** — Identified, analyzed, and recommended mitigations for organizational security risks
- **Professional Writing** — Authored management-level security briefings suitable for executive and leadership audiences
