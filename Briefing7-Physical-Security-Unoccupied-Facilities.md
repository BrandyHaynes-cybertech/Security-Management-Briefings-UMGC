# Security Briefing — Physical Security for Unoccupied Facilities
**Course:** CMIT 425 — Cybersecurity Management | UMGC  
**Author:** Brandy Haynes  

---

## Overview

Providing physical security in an unoccupied or unattended facility poses significant challenges — especially during events like the COVID-19 pandemic, which forced businesses to scale down in-person staffing. This briefing addresses preventative and detective controls for securing facilities and assets without requiring full staff presence on-site.

---

## Access Control Levels

### Mandatory Access Controls (MAC)
- Restrict access based on security clearance
- Access is dictated by the classification level of the data, files, objects, or physical spaces

### Role-Based Access Controls (RBAC)
- Provide access to company resources based on the role a user holds
- Access is granted via prior authorization methods aligned to job responsibilities

### Rule-Based Access Controls
- Utilize specific rules indicating what or who can access or modify data and files within the system
- Rules are applied uniformly regardless of user role

---

## Perimeter Security

- **Fencing** — physical barriers surrounding buildings prevent easy unauthorized access
- **IP Cameras** — have replaced older CCTV systems with:
  - Increased coverage area
  - Greater pixel detail in standard, playback, and zoom modes
  - Greater scalability than analog CCTV models
- **Lighting** — proper illumination in walkways, alleys, parking lots, and garages deters bad actors
- **Motion detectors, sensors, and alarms** — detect unauthorized movement and trigger alerts
- Proper placement of all detection and monitoring equipment is critical to eliminating blind spots

---

## Controlled Zones

Assign specific security zones throughout the facility based on the activities conducted in each area:

| Zone | Access Level |
|------|-------------|
| Front Lobby | Public access — high traffic |
| Executive Offices | Restricted — executives only |
| Product Development | Top secret — development personnel only |
| Server Rooms / Data Centers | Restricted — IT personnel only |
| Valuable Asset Storage | Strict security — limited authorized personnel |

---

## Work Area Separation

- All personnel can access the front lobby
- Only authorized individuals should access restricted areas to prevent theft or tampering
- IT personnel are the only staff with access to server rooms
- Executives and managers are the only personnel with access to secured or executive office areas
- Product development personnel have exclusive access to development zones

---

## Technological Physical Security Controls

Implement one or more of the following technical controls:

- **Firewalls** — network-level protection preventing unauthorized digital access
- **Intrusion Detection Systems (IDS)** — monitors both physical and logical environments for suspicious activity
- **Biometrics** — fingerprint, retina, or facial recognition for high-security access points
- **Smart or Dumb Cards** — physical access credentials for badge-controlled entry
- **Proximity Readers** — contactless access control at secured entry points
